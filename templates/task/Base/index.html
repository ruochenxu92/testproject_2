Introduction to Super Quick Query(superqq) Website

The website is the intelligent web system mainly for recommending search paper for professors. The data comes from Google Scholar and Arxiv.org and updated daily using scrapy framework. Currently, we focus on how to improve the search quality and match. I prefer using BM25. The project mainly functionalities will finish March 29th.
